# JustHTML

## **Usage** ##
Add the two lines to your html code: <br />
  >`<link href="https://aquitano.github.io/Other/JustHTML/JustHTML.css" rel="stylesheet"/>` <br />
  >`<script src="https://aquitano.github.io/Other/JustHTML/JustHTML.js"></script>`
